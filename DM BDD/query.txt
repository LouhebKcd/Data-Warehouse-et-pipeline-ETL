1.
Question :
Quels matériels appartenant à l'entreprise ETS_KACED sont utilisés sur quels chantiers, et qui sont les chauffeurs associés ?

Requête :
PREFIX vl: <http://www.semanticweb.org/kaced211/vehicules#>
SELECT ?entreprise (?materiel AS ?Vehicule) ?chantier ?chauffeur
WHERE {
    ?materiel vl:appartientALentreprise ?entreprise .

    OPTIONAL {
        ?materiel vl:travailleSurChantier ?chantier .
    }

    OPTIONAL {
        ?materiel vl:aPourChauffeur ?chauffeur .
    }

    FILTER(?entreprise = vl:ETS_KACED)
}

Question :
Quels sont les véhicules de chaque constructeur avec leurs caractéristiques (moteur, transmission et consommation) ?

Requête :
PREFIX vl: <http://www.semanticweb.org/kaced211/vehicules#>
SELECT ?constructeur ?vehicule ?moteur ?transmission (STR(?consommation) AS ?consommationMoyenne)
WHERE {

    ?vehicule rdf:type vl:VehiculeLeger .
    ?vehicule vl:aPourConstructeur ?constructeur .

    OPTIONAL {
        ?vehicule vl:aPourMoteur ?moteur .
    }
    OPTIONAL {
        ?vehicule vl:aPourTransmission ?transmission .
    }
    OPTIONAL {
        ?vehicule vl:aPourConsoMoyEnLitreAu100 ?consommation .
    }
}
ORDER BY ?constructeur
